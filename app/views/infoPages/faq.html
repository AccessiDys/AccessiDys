Foire aux questions

MES PROFILS


Comment accéder aux profils ?

  • Ouvrir le Menu burger en haut à droite de votre écran.
  • Cliquer sur l’onglet « Mes profils », une nouvelle page va s’ouvrir.
  • Pour voir tous les profils, scroller (faire défiler verticalement) vers le bas la page.
  • Vous pouvez également masquer l’aperçu des profils en cliquant sur le bouton « MASQUER L'APERÇU DES PROFILS », dans cette configuration vous avez la liste de vos profils et des profils par défaut qui s’affichent.
  • Pour afficher l’aperçu d’un profil vous il vous suffit de cliquer sur la petite flèche à gauche du profil.

Comment créer un nouveau profil ?

  • Accéder à la page MES PROFILS par le menu.
  • Cliquer sur le bouton « AJOUTER UN PROFIL » en haut à droite, une nouvelle fenêtre s’ouvre.
  • Renommer votre profil, paramétrer la synthèse vocale, modifier les styles en cliquant sur les crayons.
  • Valider vos modifications en cliquant sur « ENREGISTRER LE PROFIL ».

Comment paramétrer la synthèse vocale ?

  • Vous pouvez régler le volume, la vitesse, la hauteur de voix.
  • Par défaut la synthèse vocale est programmée en français
  • Vous pouvez choisir la langue (Français, Anglais, Allemand, Espagnol), très utile si vous avez besoin de travailler sur un texte d’une langue étrangère. Nous vous conseillons de faire un profil par langue, pour vous faciliter l’adaptation et l’utilisation de la synthèse vocale.
  • Valider vos modifications en cliquant sur « ENREGISTRER LE PROFIL ».

Comment modifier les valeurs du style ?

  • Pour chaque style, (titre 1, sous-titre, paragraphe…) vous pouvez paramétrer les styles : polices, taille, interligne, coloration, graisse, espaces entre les mots et entre les caractères.
  • Pour vous faciliter les réglages des styles, vous avez la possibilité d’appliquer les paramètres du style que vous venez de régler à tous les styles du profil, en cochant la case.
  • Valider vos modifications en cliquant sur « ENREGISTRER LE PROFIL ».

Comment modifier un profil déjà créé ?

  • Allez sur la page Mes Profils
  • Cliquer sur la flèche descendante (dans la colonne action), dans ce sous menu vous avez la possibilité de modifier, dupliquer, déléguer, partager, supprimer, et ajouter à mes favoris le profil sélectionné.
  • Sélectionner modifier, une nouvelle fenêtre s’ouvre avec les informations liées au profil, cliquer sur les crayons pour apporter les modifications.

Pourquoi déléguer un profil ?

Cette fonction vous permet de déléguer votre profil à un professionnel de santé qui vous suit (ergothérapeute,…).
Ce professionnel, recevra un mail avec un lien lui permettant de paramétrer votre profil à vos besoins.


Pourquoi partager un profil ?

Vous êtes professionnel de la santé, ou utilisateur, et vous pensez que votre profil que vous venez de créer peut être utile à une autre personne.
Vous souhaitez le partager, vous avez la possibilité de le faire soit par mail, soit via les réseaux sociaux.


MES DOCUMENTS


Comment ajouter un document ?

  • Allez sur la page Mes DOCUMENTS
  • Cliquer sur le bouton « AJOUTER UN DOCUMENT », vous vous retrouvez dans l'éditeur.
  • Soit vous créer votre document, soit vous ouvrer un document sous format Word.docx, PDF, Epub.
  • Si votre format de document n'est pas compatible, vous avez un message d'alerte qui apparait, vous avez malgré tous la possibilité d'ouvrir votre document dans un logiciel et de faire un copier-coller dans l'éditeur.

Comment créer ou modifier un document

L'éditeur de texte vous permet de créer un document, votre texte s'adaptera en simultané en fonction du profil.
Vous pouvez également télécharger des fichiers word .docx.
N'oubliez pas d'enregistrer votre document, ainsi vous le retrouverez facilement dans la page "MES DOCUMENTS" avec la possibilité de le modifier, de le partager, de le supprimer.


Comment adapter un pdf

Un document au format pdf, est une image, pour adapter le texte il va falloir faire une RECONNAISSANCE OPTIQUE DE CARACTÈRES.
L'application va télécharger votre pdf dans l'éditeur mais rien ne se passera, en cliquant sur l’image une petite fenêtre apparait vous devez cliquer sur le bouton reconnaissance de caractères.
Une autre fenêtre s’ouvre où vous pouvez faire un sélection sur le pdf d’une partie de texte ou la page entière puis cliquer sur « LANCER LA RECONNAISSANCE DE CARACTERES ».
Votre sélection sera adaptée dans l’éditeur, et votre profil appliqué.


WEB ADAPTÉ


Comment installer le bouton « AccessiDys » ?

  • Ouvrir le Menu burger en haut à droite de votre écran.
  • Cliquer sur l’onglet « Web adapté », une fenêtre va alors s’ouvrir.
  • Sélectionner le bouton AccessiDys et glissez-le dans la barre de favoris de votre navigateur.

Comment utiliser le web adapté ?

  • Ouvrir la page internet dans votre navigateur que vous souhaitez lire.
  • Cliquer sur le bouton « AccessiDys » installé dans votre navigateur, la page va s’afficher dans l’application.

Est-il possible de naviguer sur le web via AccessiDys ?

Oui. Une fois qu’une page web est ouverte via AccessiDys, vous pouvez continuer à naviguer sur le web.
Lorsque vous cliquez sur un lien, celui-ci va s’ouvrir dans une nouvelle page.


Peut-on changer de profil lorsqu'on utilise le web adapté ?

Oui. Vous pouvez adapter les pages Web aux différents profils que vous avez créés ou ceux déjà existants.
Pour changer le profil actuel, cliquer sur la flèche descendante puis sélectionner le profil que vous souhaitez.


SAUVEGARDE

Comment sauvegarder vos documents et profils ?

  • Allez sur la page sauvegarde via le menu burger en haut à droite,
    NB: si vous avez un triangle rouge qui apparaît faite la procédure suivante.
  • Connectez-vous à un espace de stockage
    • Sélectionner Dropbox en cliclant sur le logo
    • Poursuivre en clicquant sur le bouton s'identifier si vous avez déjà un compte Dropbox ou sur le bouton créer et suivre la procédure.
    • Le bouton "Annuler" vous permet de revenir vers la page vos documents sans avoir fait d'enregistrement sur un espace de stockage.
      Attention dans ce cas vos documents et profils ne seront enregistrés que dans le cache de votre navigateur. Si vous vider votre cache, ou si vous souhaitez utiliser l'application sur un autre support (tablette) vous ne retrouverez pas vos documents et profils sauf si vous avez effectuer une sauvegarde sur un espace de stockage.

Comment se déconnecter de votre espace de sauvegarde ?

Tous simplement clicler sur déconnecter-vous, cette action vous déconnectera de votre espace de stockage et videra le cache du navigateur, très utilse si vous utiliser un ordinateur multi utilisateur.
Si vous traviller sur votre ordinateur personne, une fois la première connection faite vous n'avez plus besoin de faire cette procédure, vos documents et profils seront sauvegarder. Vous pouvez traailler hors connexion internet et dès que vous retrouvez une connexion l'application mets à jour votre sauvegarde sur l'espace de stockage.

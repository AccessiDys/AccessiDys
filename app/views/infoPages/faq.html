FAQ

Les profils

Comment accéder aux profils ?

  • Ouvrir le Menu burger en haut à droite de votre écran.
  • Cliquer sur l’onglet « Mes profils », une nouvelle page va s’ouvrir.
  • Pour voir tous les profils, scroller (faire défiler verticalement) vers le bas la page.

Comment créer un nouveau profil ?

  • Accéder aux profils (lien vers la question)
  • Cliquer sur le bouton « Ajouter un profil » en haut à droite, un nouvel encart va s’ouvrir.
  • Renommer votre profil, modifier les styles en cliquant sur les crayons.
  • Valider vos modifications en cliquant sur « Enregistrer le profil ».

Est-il possible de modifier un profil déjà créé ?

  • Ouvrir la page (lien vers la question) avec tous les profils.
  • Cliquer sur la flèche descendante (dans la colonne action).
  • Sélectionner modifier, un nouvel encart va s’ouvrir.
  • Pour modifier le nom du profil :
    • 1) Cliquer sur le stylo dans la partie « Informations liées au profil ».
    • 2) Modifier le nom puis valider avec le bouton « OK ».
  • Pour modifier les styles du profil
    • 1) Cliquer sur le stylo correspondant au style à modifier.
    • 2) Modifier chacun des paramètres puis valider avec le bouton « Enregistrer le style ».
    • 3) Répéter les deux précédentes étapes pour chacun des styles.
  • Valider vos modifications en cliquant sur « Enregistrer le profil ».

Le Web adapté


Comment installer le bouton « AccessiDys » ?

  • Ouvrir le Menu burger en haut à droite de votre écran.
  • Cliquer sur l’onglet « Web adapté », une fenêtre va alors s’ouvrir.
  • Sélectionner le bouton AccessiDys et glissez-le dans la barre de favoris de votre navigateur.

Comment utiliser le web adapté ?

  • Ouvrir la page que vous souhaitez lire.
  • Cliquer sur le bouton « AccessiDys » installé dans votre navigateur, la page va s’afficher dans l’application.

Est-il possible de naviguer sur le web via AccessiDys ?

Oui. Une fois qu’une page web est ouverte via AccessiDys, vous pouvez continuer à naviguer sur le web. Lorsque vous cliquez sur un lien, celui-ci va s’ouvrir dans une nouvelle page.


Peut-on changer de profil lorsque l’on utilise le web adapté ?

Oui. Vous pouvez adapter les pages Web aux différents profils que vous avez créés ou ceux déjà existants.
Pour changer le profil actuel, cliquer sur la flèche descendante puis sélectionner le profil que vous souhaitez.

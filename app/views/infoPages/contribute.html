Contribuer

Accessidys est un projet pensé comme un commun, dont le logiciel est libre. L'ensemble des utilisateurs et partenaires d'Accessidys peuvent contribuer à l'amélioration de ce projet, autant au niveau technique que sur les autres aspects. Voici ci dessous divers moyens de participer. Si vous avez d'autres idées, n'hésitez pas à nous contacter

Contribuer à Accessidys

Liste de discussion | Actions en cours | Slack | Contact | Github |

  • Accéder au code source / Forker le projet
  • Installez votre propre serveur AccessiDys à l'aide de la documentation
  • Rejoignez la liste de discussion et envoyez nous sur le mail de contact une demande pour rejoindre le Slack ou le Trello.
  • Signalez une anomalie ou une évolution en remplissant le formulaire sur github
